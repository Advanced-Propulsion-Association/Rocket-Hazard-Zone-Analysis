# V2 — 6-DOF Hazard Zone Calculator Design Spec

**Date:** 2026-04-06  
**Status:** Approved  

---

## Overview

Version 2 is a full 6-DOF (six degrees of freedom) hobby rocket hazard zone calculator. It is a standalone web app that lives alongside v1 in the same repository. Where v1 uses a conservative 3-DOF point-mass model to produce a worst-case bounding radius, v2 uses a physically accurate 6-DOF simulation with Monte Carlo dispersion to produce a statistically grounded hazard footprint.

## Input

### Primary: OpenRocket .ork Import

The `.ork` file is the primary and preferred input method. The extended `orkParser.ts` reads:

- Body geometry: diameter, length
- Nose cone: type and length
- Fin geometry: root chord, tip chord, span, sweep
- Component masses and CG positions (per stage)
- Barrowman CP location
- Motor designation (for ThrustCurve.org auto-lookup)
- Max apogee from OR simulation (for comparison panel)

Fields still estimated from geometry (not available in .ork):
- Roll damping coefficient (Clp)
- Pitch/yaw damping coefficients (Cmq, Cnr)
- Initial roll rate (assumed 0 unless overridden)

### Fallback: Manual Entry

All geometry fields available as manual inputs, mirroring v1 Tier 2/3. Shown in a collapsible panel below the .ork drop zone. Motor entry supports ThrustCurve.org lookup, RASP `.eng` upload, and boxcar (average thrust + burn time).

### Advanced Overrides (collapsible)

Optional expert panel exposing:
- Axial moment of inertia Ixx (kg·m²)
- Transverse moment of inertia Iyy (kg·m²)
- CNα — normal force coefficient slope (1/rad)
- Cmq — pitch damping coefficient
- Cnr — yaw damping coefficient
- Clp — roll damping coefficient
- Initial roll rate p₀ (rad/s)

If not provided, all are estimated from geometry.

## Physics Engine

### State Vector (13 variables)

```
Position:       x, y, z          (m — x downrange, y lateral, z altitude AGL)
Velocity:       vx, vy, vz       (m/s)
Euler angles:   φ (roll), θ (pitch), ψ (yaw)   (rad)
Angular rates:  p (roll rate), q (pitch rate), r (yaw rate)   (rad/s)
Mass:           m                (kg)
```

### Equations of Motion

**Translational** (same as v1 but 3D):
- Forces: thrust (along body axis), aerodynamic drag + normal force, gravity
- Normal force computed from CNα × angle of attack × dynamic pressure × reference area

**Rotational** (new in v2 — Euler's equations):
```
Ixx · ṗ = L  (roll moment — fin cant, roll damping Clp)
Iyy · q̇ = M  (pitch moment — CNα restoring + Cmq damping)
Iyy · ṙ = N  (yaw moment — CNα restoring + Cnr damping)
```

**Kinematic** (Euler angle rates from body angular rates):
```
φ̇ = p + (q·sin φ + r·cos φ)·tan θ
θ̇ = q·cos φ − r·sin φ
ψ̇ = (q·sin φ + r·cos φ) / cos θ
```

### Aerodynamic Coefficients

- **CD:** carried over from v1 fineness-ratio model + Mach correction
- **CNα:** from Barrowman equations using fin + body geometry (or from .ork, or manual override)
- **Damping:** Cmq and Cnr estimated as `−2 × CNα × (reference length / 2) / diameter`; Clp estimated from fin planform area and moment arm. All overridable.

### Integrator

RK4, adaptive timestep: 0.01 s during burn, 0.05 s coasting. Terminates when `z < −1.0 m` after rocket has been above 5 m. Same safety bail as v1 (no-thrust escape after 30 s).

## Monte Carlo Dispersion

Runs entirely in a **Web Worker** to keep the UI responsive. User selects N via a slider: 100 / 500 / 2000. Progress is reported back to the main thread every 50 runs and shown in a progress bar.

### Randomized Parameters (per run, independent uniform distributions)

| Parameter | Variation |
|---|---|
| Wind speed | ±20% of nominal |
| Wind direction | 0°–360° uniform |
| Launch angle magnitude | ±2° from nominal |
| Motor total impulse | ±3% (manufacturing tolerance) |
| Initial pitch perturbation | ±1° |

### Output

Each run produces a single landing point (x, y) in meters from the launch pad. The full scatter cloud is plotted as a 2D top-down footprint. The hazard radius is the radius of the smallest enclosing circle of all landing points (99th-percentile radius as an alternative display option).

---

## Outputs

### 1. Hazard Zone Radius
- Max enclosing circle radius of Monte Carlo scatter (ft and m)
- Displayed as a large result card, same style as v1

### 2. 2D Scatter Plot (Plotly)
- Top-down view of all N landing points
- Enclosing circle overlaid
- Color-coded by run index (cool-to-warm gradient shows spread over run order)
- Launch pad marked at origin

### 3. Attitude History Plot (Plotly)
- Nominal trajectory (center-case Monte Carlo run)
- Three lines: pitch angle θ, yaw angle ψ, roll angle φ vs time
- Angle of attack α overlaid as a dashed line
- Red zone shading above α = 15° (aerodynamic stall / instability threshold)

### 4. 3-DOF vs 6-DOF Comparison Panel
- Side-by-side: v1 hazard radius vs v2 hazard radius
- Max apogee comparison
- Short explanation of why results differ (6-DOF accounts for attitude dynamics; v1 is intentionally conservative)

## Physics Review Gate

Before v2 results are published or used for any regulatory purpose, the physics engine must be reviewed by a dedicated aerodynamics review agent. The review should verify:

1. Barrowman CNα implementation against the original Barrowman (1967) equations
2. Euler rotation equation sign conventions and coordinate system consistency
3. Kinematic equation singularity handling near θ = ±90° (gimbal lock)
4. Damping coefficient estimation formulas against published references
5. Monte Carlo parameter distributions against FAA/range safety precedent
